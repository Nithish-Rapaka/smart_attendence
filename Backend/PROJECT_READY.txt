## 🆘 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| PostgreSQL won't connect | Check PostgreSQL is running on localhost:5432 |
| Port 8080 in use | Change to 8081 in application.properties |
| Build fails | Run `.\mvnw clean dependency:purge-local-repository` |
| Tests fail | Ensure PostgreSQL database exists |
| JWT errors | Verify token format and expiration |
